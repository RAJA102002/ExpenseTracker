# 💸 Expense Tracker (Java)

A simple Java console application to manage daily expenses.

## 🚀 Features
- Add expenses
- View all expenses
- Calculate total
- Save data to file

## 🛠️ Tech Used
- Java
- File Handling
- OOP

## ▶️ Run in Eclipse
1. Import project
2. Run ExpenseTracker.java
